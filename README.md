# telegrambotmusic
бот для облегченной навигации и поиска сохраненной музыки.
также имеет возможность сохранить текст песни и выдавать по запросу - выдает по нескольку строк за раз.
все песни нужно отправлять боту самостоятельно - можно через другие боты-поисковики музыки - например - @mus_vir_bot
